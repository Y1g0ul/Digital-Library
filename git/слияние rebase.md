При ребейзе [[git]] по сути копирует набор [[коммит]]ов и переносит их в другое место. C его помощью можно делать чистые и красивые линейные последовательности [[коммит]]ов.

`git rebase [branchname]` - «отматывает» ваши коммиты из ветки,  применяет все новые коммиты из branchname и поверх них применяет ваши коммиты заново.

![[git rebase]]

- `-i` - опция для открытия интерфейса с целью просмора того какие [[коммит]]ы готовы к копированию. Так же показывает хеши [[коммит]]ов и комментарии к ним. ( `git rebase -i HEAD~4` )
