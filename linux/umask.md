Для файлов в [[Linux]] существует атрибут по умолчанию.

Для файлов — `644`, для директорий — `755`.
Берём некую маску, которую вычитаем из `666` для файлов и `777` для директорий. По умолчанию это `022`. Таким образом, `666` – `022` = `644`, `777` – `022` = `755`. Задается командой  `umask 022`
