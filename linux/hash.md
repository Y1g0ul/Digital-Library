Команда в [[Linux]] которая показывает какие команды использовались и хешировались и сколько попыток было из использовать. При первом запуске команды она берется из файла а потом попадает в кеш и берется уже из него.

``` bash
y1g0ul@DESKTOP-195S4U1:~$ hash
hits    command
   1    /usr/bin/env
   2    /usr/bin/ls
   1    /usr/bin/mkdir
   3    /usr/bin/whereis
   1    /usr/bin/rm
```

`hash -r` - очистка кеша

В большинстве случаев команды - это просто внешние исполняемые файлы.
