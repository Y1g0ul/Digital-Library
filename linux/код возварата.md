| код   | значение                                                      |
| ----- | ------------------------------------------------------------- |
| `0`   | Успех                                                         |
| `1`   | Общая ошибка                                                  |
| `2`   | Неправильное использование команды (например, файл не найден) |
| `127` | Команда не найдена                                            |
Когда ты запускаешь команду в [[Linux]] (например, [[ls]] [[cat]], [[mkdir]]), она завершает свою работу и сообщает системе:   
- "Всё прошло успешно"(обычно код `0`)  
- "Произошла ошибка"(код от `1` и выше, в зависимости от типа ошибки).  

Что делает `echo $?`?
- `$?` — это специальная переменная, которая хранит код возврата последней выполненной команды.  
- `echo $?` просто выводит этот код на экран.
