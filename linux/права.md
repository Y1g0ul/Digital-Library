Права в [[Linux]] делсятся по умолчанию на 3 типа:
- **r (read - чтение):** Просмотр содержимого файла или списка файлов в каталоге.
- **w (write - запись):** Изменение файла или создание/удаление файлов в каталоге.
- **x (execute - выполнение):** Запуск исполняемого файла или вход в каталог.
(Они не исключают друг друга, может быть только одно, а может и все три сразу)


-==rw-==*r--*`r--` 1 y1g0ul y1g0ul   11 Dec  9 10:03 file

- ==rw-== - права для пользователя который является владельцем этого файла
- *r--* - права для того кто входит в основную группу владельца
- `r--` - права для всех остальных

Если наш файл представляет из себя каталог - `drwxr-xr-x 2 y1g0ul y1g0ul 4096 Dec  9 10:04 dir`  то права `x`(execute) означают возможность зайти в этот каталог
