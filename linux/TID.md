Thread ID - уникальный номер, который ядро [[Linux]] присваивает каждому [[поток]]у внутри системы.

```bash
y1g0ul@LAPTOP-MSSNKT4E:~$ ps -T -p 1
  PID  SPID TTY          TIME CMD
    1     1 ?        00:00:00 init
    1     7 ?        00:00:00 init
```

В `ps -T -p <PID>` (просмотр [[поток]]а конкретного [[процесс]]а) колонка `SPID` показывает идентификатор потока, который в ядре называется TID.    
Если [[поток]] — основной (первый) в процессе, то его TID = [[PID]].
Остальные [[поток]]и имеют TID ≠ [[PID]], но [[PID]] у них остаётся «родительским» для [[процесс]]ов.
