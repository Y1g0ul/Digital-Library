head:
- `head myfile` - команда в [[Linux]] которая выводит **первые** 10 строк файла.
- `head -3 myfile` - **первые** 3 строки файла
- `head -n 3 myfile` - так же можно пользоваться **ключами**

wc:
- `wc myfile` - выводит **количество** строк, слов и букв.
- `-l`  - считает количество строк
- `-w` -  считает количество слов
- `-c`  - считает количество символов

tail:
- `tail myfile` - то же самое что и head только о **нижних** строчках
- `-f`  - не закрывает файл и **обновляет** информацию

grep:
- `grep request myfile` - **выведет** все строки из файла содержащие request
- `-v` - выведет все строки из файла не содержащие request
- `-i` - для **игнорирования** регистра
- `-r ` - для рекурсивной работы
