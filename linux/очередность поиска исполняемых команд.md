В [[Linux]] при вводе команды происходят эти дейсвия:
1. bash проверяет не является ли введённая команда его внутренней командой.
``` bash
myuser@ruvds-baz2e:~$ type cd
cd is a shell builtin
```
2. Просмотр алиасов. ( В командном интерпритаторе bash можно за определенным названем закрупить какую-нибудь другую команду. )
``` sh
myuser@ruvds-baz2e:~$ type ls
ls is aliased to 'ls --color=auto'
```

3. Используется переменная [[$PATH]]. Bash начинает перебирать каталоги используемые в этой переменной. (Будет выполнен первый найденный файл равный по имени имени команды)

> [!info] Результат выдается по первому найденному совпадению
> То есть если у нас есть 2 файла с одинаковым именем в разных каталогах то выдастся только один, который найдется первым.
